About

Peercoin Solo Pool is not a conventional mining pool. Mining shares are not pooled, and block rewards are immidiately mined to the miner's address. Meaning that the payments show immediately in the miner's wallet.

Connect your miner and get:

  • Immediate reward payouts.
  • Optionally mine straight to a P2SH multisig address.
  • Configurable pool donation, used for peercoin development.
  • [Coming soon] Option to donate part of your block reward to trusted projects.

Puzzles

Puzzles are funds locked by a script puzzle. One can lock a certain amount of Peercoins that can be spent by anyone knowing the solution to: x + 5 = 6.
This section contains a set of puzzles that you can solve to earn some Peercoins.
After solving a puzzle, we kindly ask you to create a new puzzle for other people to solve and add it to this project by creating a pull request on github.

    Smart Contracts

    Smart Contracts are used to lock funds under certain conditions. Choosing these conditions wisely allows for the creation of specialized trustless contracts.
